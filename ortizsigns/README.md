# Ortiz Signs (React)

Sitio migrado a React + Vite.

## Comandos

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run preview`

## Panel del dueño

- Botón: `Panel del dueno` (esquina inferior izquierda)
- PIN inicial: `2026`
- Permite editar foto, textos y contacto.
- La información se guarda en `localStorage` del navegador.
